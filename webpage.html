This is the beginning of our web projects. We're pushing our code in our github repository through gitbash. We are learning about web development, IDE and languages used and it's scope. Along with it we are learning about Github and it's use.

Command to push file in git repository:
  • git init
  • git config user.name "name"
  • git config user.email "email"
  • git remote add origin "link_of_your_repository"
  • git add "file-name.with .exe"
  • git add . {To select all file}
  • git commit -m "message"
  • git push origin master
  • (After edit)git push -u origin master
